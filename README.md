A simple Tic Tac Toe game where you play against the computer. The computer uses the Minimax algorithm, so it plays perfectly

it will never lose (best you can do is draw).

✨ Features- Play Tic Tac Toe against an unbeatable AI (Minimax based) Console based, simple text interface — no extra setup needed Detects win, loss, and draw correctly every time Input validation — won't crash if you enter wrong number or already filled position Clean board display after every move
